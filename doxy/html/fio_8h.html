f_io f_open ( f_io  files,
int16_t **   
)

Open input and output files which are included in the typedef f_io files.

Only use this function with a partly intialised f_io struct .in_id and .outid aswell in_mode and out.mode must be well defined or else the program will have undefined behavior.

Returns in each input and output .in , .out a valid FILE* value upon success or it exits the program upon fopen failure.

Each f_io type struct that opens with f_open must be closed with f_close.
