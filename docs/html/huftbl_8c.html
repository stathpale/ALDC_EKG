Detailed Description

Static Huffman Tables.

author: Palaiologos Efstathios

This .c file contains two different sets of Huffman tables, each one with three discrete levels. The identifiers huf_tblx and huf_tblxb, stand for set a and b respectively, while x represents the level. For each Huffman table there is corresponding table that contains the length of the unique prefix code of each element.
