Detailed Description

Paths of the datasets.

author: Palaiologos Efstathios

Defines two char const*[] objects, the first has the paths of the selected original dataset(uncompressed) and the other has the paths the corresponding cocompressed dataset.

define exactly one of the macros MIT_DATASET and __CUSTOM_DATASET___
