Detailed Description

File input and output Interface.

author: Palaiologos Efstathios This module provides an opaque interface for opening, loading and closing of continous time recrodings. It also provides an opaque interface for creating new files.

f_io f_open ( f_io  files,
int16_t **  difbuf_adr

Opens input and output files which are included in the typedef f_io files.

Each f_io type struct that opens with f_open must be closed with f_close.

Parameters
[in]files.in_id and .out_id must be defined prior to the invocation of this function or else the program will have undefined behavior.

[out]files.ina valid FILE* pointer at the input file
[out]files.outa valid FILE* pointer at an empty new file

[out]difbuf_adrthe address of the allocated memory for the input buffer, filled with the differences of all the sequential pairs of samples.
